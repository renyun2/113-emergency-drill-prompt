用 Go + Vue3 + PostgreSQL 构建一个应急预案与演练记录管理系统，Docker Compose 部署。

技术栈：
- 后端：Go (Gin + GORM)
- 前端：Vue3 + Vite + Element Plus
- 数据库：PostgreSQL 16
- 部署：Docker Compose

核心功能：
1. 预案管理：录入应急预案（预案名称/类型：火灾/危化品泄漏/地震/停电/自然灾害/适用场景/预案版本/发布日期/修订记录/文件内容 Markdown 格式），预案版本管理，旧版本存档可查
2. 预案审批：预案修订后提交审批（编制人/编制日期/审批人/批准日期），审批通过后设为当前生效版本
3. 演练计划：创建演练计划（预案名称/演练类型：桌面推演/实战演练/联合演练/计划日期/地点/组织部门/参与人员范围/演练目标），提前通知相关部门
4. 演练记录：演练完成后录入演练记录（实际参与人数/演练时长/演练过程描述/发现问题清单/评估结论：优秀/良好/一般），上传演练照片路径
5. 问题整改：演练发现的问题生成整改项（问题描述/责任人/整改措施/完成期限），跟踪整改落实，形成闭环
6. 合规统计：本年已开展演练次数，各预案演练频次是否满足要求（如每年至少1次），演练覆盖部门比例，整改完成率

数据表：emergency_plans、plan_versions、drills、drill_issues、rectifications
init.sql 预置10份应急预案和2年历史演练记录。
docker compose up -d 后访问 http://localhost:8080。
